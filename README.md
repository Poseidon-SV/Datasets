# Datasets.git
Large Datasets/file github upload commands 

## git LFS
### Open gitbash in that folder 

`git lfs install`

`git lfs track "*.csv"`

`git add .gitattributes`

`git add file_name.csv`

`git commit -m "commit"`

`git lfs ls-files`

`git lfs migrate import --include="*.file"`

`git lfs push --all origin`

`git push origin main`
